Microsoft's cloud-native endpoint management platform. Originally built for Windows (including Windows Phone) with OMA-DM and a Silverlight frontend, but later expanded to [[Notes/Microsoft/Intune/Android|Android]], [[iOS|iOS]], macOS, and a dedicated Windows agent (Intune Management Extension).
Also supports [[BYOD|BYOD]] via their own SDK called MAM.
